# 🎬 CinemaTime

Welcome to **CinemaTime**, an ultra-fast, visually stunning movie, TV show, and anime recommender system powered by Artificial Intelligence!

[![Live Demo](https://img.shields.io/badge/Live_Demo-cinema--time--black.vercel.app-00b4ff?style=for-the-badge&logo=vercel)](https://cinema-time-black.vercel.app)

## ✨ Features
- **Massive Database:** Thousands of movies, TV shows, and anime fetched natively from the TMDB API.
- **AI-Powered Search:** Built with a hyper-optimized FAISS vector database and `SentenceTransformer` embeddings for lightning-fast, semantic similarity recommendations.
- **Stunning UI:** A dark-mode, glassmorphism-inspired Next.js frontend with smooth Framer Motion micro-animations.
- **Cinematic Experience:** Auto-playing, borderless YouTube trailers embedded right into the hero sections.
- **Dynamic Categories:** Seamlessly flip between Movies, TV Shows, and Anime, complete with dynamically generated genre rows.

## 🚀 Live Application
You don't need to run this locally anymore! The full application is hosted in the cloud and accessible 24/7 on any device.

## 🛠️ Tech Stack
**Frontend:**

- Framer Motion

**Backend:**
- [FastAPI](https://fastapi.tiangolo.com/) (Python)
- [FAISS](https://faiss.ai/) (Facebook AI Similarity Search)
- Sentence-Transformers
- Pandas / NumPy
